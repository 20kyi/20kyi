<h3>🌿  MODI (Currently Building)</h3>

> 일상의 모든 순간을 가치 있게 수집하다.

소소한 데일리 미션을 통해 사용자가 의미 있는 습관을 형성할 수 있도록 돕는 아름다운 디자인의 일과 관리 앱입니다.

<h4>Tech Stack</h4>

`SwiftUI` `Node.js` `PostgreSQL` `AWS` `Railway`

<h4>Key Features</h4>

- ✨ 데일리 미션 제공
- 📸 나만의 컬렉션 수집
- 🎥 애니메이션이 포함된 컬렉션 공유 기능
- 📅 한눈에 보는 달력(캘린더)
- 🔥 꾸준함을 기록하는 스트릭(Streak)
- ☁️ 안전한 클라우드 동기화

---

<h3>📍  MemoWay</h3>

> 길 위에서 마주하는 소중한 기억과 기록의 흔적.

내가 지정한 특정 장소에 도달했을 때 알림을 보내주고, 지도 위에 일상의 소중한 메모를 기록하고 보관할 수 있는 **위치 기반 메모 및 리마인더 앱**입니다.

<h4>Tech Stack</h4>

`Flutter` `Dart` `PostgreSQL` `Railway`

<h4>Key Features</h4>

- 📍 **위치 기반 알림 (Geofencing)** : 특정 장소 근처에 도착하면 잊지 않도록 스마트한 알림 제공
- 🗺️ **지도 위에 그리는 일상** : 내가 남긴 일기와 순간의 메모들을 지도 위에 핀으로 시각화
- ✍️ **간결하고 깔끔한 다이어리** : 복잡함 없이 매일의 생각과 기록을 담백하게 남길 수 있는 에디터
- ☁️ **안전한 실시간 데이터 보관** : PostgreSQL 데이터베이스를 활용한 안정적인 데이터 동기화

---

<h3>🧪  StoryPotion</h3>

> 평범한 일상을 마법 같은 한 편의 소설로.

매주 작성한 일상 다이어리들을 인공지능(AI)이 분석하여, 판타지나 로맨스 등 유저가 원하는 장르의 흥미진진한 피클 소설로 재탄생시켜 주는 **스토리 변환 다이어리 앱**입니다.

<h4>Tech Stack</h4>

`React` `Capacitor` `Node.js` `Express` `Firebase`

<h4>Key Features</h4>

- 🪄 **일기의 소설화 (AI 변환)** : 한 주 동안 쌓인 일상 기록을 매끄럽고 몰입감 넘치는 소설 스토리로 빌드
- 🎨 **감성적인 인터페이스** : 따뜻하고 신비로운 물약 테마의 UI/UX 디자인
- 📚 **나만의 책장** : 변환 완료된 소설들을 에피소드별로 차곡차곡 보관하고 언제든 다시 읽을 수 있는 서재 기능
- 🔒 **프라이버시 및 동기화** : 소중한 개인 기록을 안전하게 서버에 동기화하고 보호
